Loan data was analyzed to predict whether a loan is fully paid using machine learning. The dataset was explored and visualized, and categorical data was converted into numerical values for modeling.

Two models were used: Decision Tree and Random Forest. Both showed high accuracy for fully paid loans but low recall for the minority class, indicating class imbalance. The confusion matrix also showed many minority cases were misclassified.

The Random Forest performed better than the Decision Tree because it reduces overfitting and gives more reliable predictions.
